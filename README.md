#### How to Play
‚ûù  Roll the dice until every die shows the same number. <br />
‚ûù  Click on a die to hold its number in place while you re-roll the rest. <br />
‚ûù  Win when all dice are held and match. <br />

## Features

- Click any die to hold or release it
- Roll only unheld dice in each turn
- Clear visual feedback for held dice
- Confetti celebration when all dice match üéâ

## Steps to Run this Project Locally

**Ensure that **Node.js** and **npm** are installed.

```bash
git clone https://github.com/meghavx/play-tenzi.git
cd play-tenzi
npm install   
npm run dev
```

##### The app will run at http://localhost:5173 by default.
